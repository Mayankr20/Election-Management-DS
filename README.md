# Election Result Management System
A Data Structures project built in C for my 1st Semester B.Tech (CSE AIML).

## Overview
This project manages an election process using **Singly Linked Lists**. 
It allows for adding candidates, casting votes, and calculating a winner dynamically.

## Data Structures Used
* **Linked List:** Used to store candidate nodes dynamically.
* **Structs:** Used to group candidate ID, Name, Party, and Vote count.

## Features
* Dynamic candidate registration.
* Search functionality to cast votes by ID.
* Automated winner calculation based on vote tallies.
